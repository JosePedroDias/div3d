createBox

(
  • dimensions
  • parentEl
  • skips
  • forEach
)

creates a 6 sided box. The order of the faces is -x, +x, -y, +y, -z, +z.

Parameters:

  • dimensions Number3

    an array of 3 integers for x, y, and z dimensions

  • parentEl DOMElement | String

    the id or element where the box gets created. defaults to the top-level node

  • skips Number

    an array of numbers ranging from 0 to 5. Each specified index gets skipped

  • forEach Function(face, index, dims)

    a callback function that gets called on each created face

createDiv

(
  • id
  • parentEl
)

creates a div object.

Parameters:

  • id String | Undefined

    an optional id to set. if falsy div3d assigns it one automatically

  • parentEl DOMElement | String

    is a DOM element or a CSS selector to it. if ommitted the div gets assigned to the root node.
